# Commands to Deploy ResourceRL to GitHub & Streamlit

## Step 1: Initialize Git (if not already done)
cd /Users/abc/Downloads/ResourceRL
git init
git branch -M main

## Step 2: Add and Commit Files
git add .
git commit -m "Initial commit: ResourceRL with Streamlit web app"

## Step 3: Create GitHub Repository
# Go to: https://github.com/new
# Repository name: ResourceRL
# Description: Autonomous Resource Scheduler using Reinforcement Learning
# Don't initialize with README
# Click "Create repository"

## Step 4: Connect to GitHub (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/ResourceRL.git
git push -u origin main

## Step 5: Deploy to Streamlit Cloud
# Go to: https://share.streamlit.io/
# Sign in with GitHub
# Click "New app"
# Repository: YOUR_USERNAME/ResourceRL
# Branch: main
# Main file: app.py
# Click "Deploy"
# Wait 1-2 minutes

## Done! Your app is live at: https://YOUR_APP_NAME.streamlit.app

## Optional: Test Locally First
streamlit run app.py
# Then visit: http://localhost:8501
